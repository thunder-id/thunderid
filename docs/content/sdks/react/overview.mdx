# React SDK

The ThunderID React SDK (`@thunderid/react`) provides a comprehensive set of components, hooks, and utilities to integrate <ProductName /> authentication and user management into your React applications.

## Installation

Install the <ProductName /> React SDK using your preferred package manager:

<CodeGroup>
  <CodeBlock lang="bash" label="npm">
    npm install @thunderid/react
  </CodeBlock>

  <CodeBlock lang="bash" label="Yarn">
    yarn add @thunderid/react
  </CodeBlock>

  <CodeBlock lang="bash" label="pnpm">
    pnpm add @thunderid/react
  </CodeBlock>

  <CodeBlock lang="bash" label="Bun">
    bun add @thunderid/react
  </CodeBlock>
</CodeGroup>

## Getting Started

To get started quickly, check out our [React Quickstart Guide](/docs/next/guides/quick-start/connect-your-application/react) for step-by-step instructions.

## Features

### Contexts

The SDK provides a powerful context provider that manages authentication state across your application.

- **[`<ThunderIDProvider />`](/docs/next/sdks/react/apis/contexts/thunderid-provider)** - Root provider component that wraps your application and manages authentication state

### Components

Pre-built, customizable components for common authentication flows and user management.

#### Action Components
- **[`<SignInButton />`](/docs/next/sdks/react/apis/components/sign-in-button)** - Trigger sign-in flow
- **[`<SignOutButton />`](/docs/next/sdks/react/apis/components/sign-out-button)** - Sign out the current user
- **[`<SignUpButton />`](/docs/next/sdks/react/apis/components/sign-up-button)** - Trigger sign-up flow

#### Control Components
- **[`<SignedIn />`](/docs/next/sdks/react/apis/components/signed-in)** - Conditionally render content for authenticated users
- **[`<SignedOut />`](/docs/next/sdks/react/apis/components/signed-out)** - Conditionally render content for unauthenticated users
- **[`<Loading />`](/docs/next/sdks/react/apis/components/loading)** - Display loading state during authentication

#### User Self-care Components
- **[`<UserDropdown />`](/docs/next/sdks/react/apis/components/user-dropdown)** - User menu with profile and settings
- **[`<UserProfile />`](/docs/next/sdks/react/apis/components/user-profile)** - Complete user profile management interface
- **[`<User />`](/docs/next/sdks/react/apis/components/user)** - Display user information

#### Organization Components (B2B)
- **[`<CreateOrganization />`](/docs/next/sdks/react/apis/components/create-organization)** - Create new organizations
- **[`<OrganizationProfile />`](/docs/next/sdks/react/apis/components/organization-profile)** - Manage organization settings
- **[`<OrganizationSwitcher />`](/docs/next/sdks/react/apis/components/organization-switcher)** - Switch between organizations
- **[`<OrganizationList />`](/docs/next/sdks/react/apis/components/organization-list)** - List user's organizations
- **[`<Organization />`](/docs/next/sdks/react/apis/components/organization)** - Display organization information

### Hooks

React hooks for programmatic access to authentication and user data.

- **[`useThunderID()`](/docs/next/sdks/react/apis/hooks/use-thunderid)** - Access the main <ProductName /> SDK instance

## Customization

All components in the <ProductName /> React SDK are fully customizable. You can:

- Override default styles using CSS classes or inline styles
- Customize component behavior through props
- Build custom components using the provided hooks
- Extend functionality with middleware and plugins
