## Run ThunderID

Start a local ThunderID instance. Pick the method that works best for you:

<RunThunderID />

Once it's running, the console is available at [https://localhost:8090](https://localhost:8090).

## Create an Application

Open the Console at <ConsoleUrl />, navigate to **Applications**, and click **Add Application**:

1. Under **Technology**, select **Vue**
2. Enter a name (e.g. `My Vue App`) and click **Create**
3. Under **Protocol**, set the **Allowed Redirect URI** to `http://localhost:5173`
4. Copy the **Client ID**. You'll need it when configuring the SDK

## Create a Vue App

Create your new Vue 3 app using Vite:

<CodeGroup>
  <CodeBlock lang="bash" label="npm">

    cd my-vue-app
    npm install
  </CodeBlock>

  <CodeBlock lang="bash" label="Yarn">
    yarn create vite my-vue-app --template vue
    cd my-vue-app
    yarn install
  </CodeBlock>

  <CodeBlock lang="bash" label="pnpm">
    pnpm create vite my-vue-app --template vue
    cd my-vue-app
    pnpm install
  </CodeBlock>
</CodeGroup>

## Register the Plugin and Add <ProductName />Provider

The `@thunderid/vue` SDK provides a Vue plugin that registers the <ProductName />Provider component globally. Register the plugin in `main.js`, then wrap your root component with <ProductName />Provider in `App.vue`.

**Step 1:** Update `src/main.js` to register the plugin:

```js title="src/main.js" showLineNumbers
import { createApp } from 'vue'
import { {{ProductName}}Plugin } from '@thunderid/vue'
import App from './App.vue'
import './style.css'

const app = createApp(App)
app.use({{ProductName}}Plugin)
app.mount('#app')
```

**Step 2:** Wrap your application with <ProductName />Provider in `src/App.vue`:

```vue title="src/App.vue" showLineNumbers
<template>
  <{{ProductName}}Provider
    client-id="<your-client-id>"
    base-url="https://localhost:8090"
  >
    <!-- your app content goes here -->
  </{{ProductName}}Provider>
</template>

<script setup>
</script>
```

:::warning Configuration
Replace `<your-client-id>` with the Client ID you obtained when creating the application in <ProductName />.
:::

### Configuration Parameters

| Parameter | Description |
|-----------|-------------|
| `client-id` | The Client ID from your <ProductName /> application |
| `base-url` | Your <ProductName /> instance URL (e.g., `https://localhost:8090`) |

## Add Sign-In and Sign-Out

The <ProductName /> Vue SDK provides `SignInButton` and `SignOutButton` components along with `SignedIn`, `SignedOut`, and `Loading` components for conditional rendering.

Replace the content of `src/App.vue` with:

```vue title="src/App.vue" showLineNumbers
<script setup>
import {
  SignedIn,
  SignedOut,
  SignInButton,
  SignOutButton,
  Loading,
} from '@thunderid/vue'
</script>

<template>
  <{{ProductName}}Provider
    client-id="<your-client-id>"
    base-url="https://localhost:8090"
  >
    <Loading>
      <div>Loading authentication...</div>
    </Loading>

    <header>
      <h1>{{ProductName}} Auth Demo</h1>
      <SignedIn>
        <SignOutButton>Sign Out</SignOutButton>
      </SignedIn>
      <SignedOut>
        <SignInButton>Sign In</SignInButton>
      </SignedOut>
    </header>
  </{{ProductName}}Provider>
</template>
```

## Display User Profile Information

Use the `User` component with a scoped slot to access and display user profile information:

```vue title="src/App.vue" showLineNumbers
<script setup>
import {
  SignedIn,
  SignedOut,
  SignInButton,
  SignOutButton,
  Loading,
  User,
} from '@thunderid/vue'
</script>

<template>
  <{{ProductName}}Provider
    client-id="<your-client-id>"
    base-url="https://localhost:8090"
  >
    <Loading>
      <div>Loading authentication...</div>
    </Loading>

    <header>
      <h1>{{ProductName}} Auth Demo</h1>
      <SignedIn>
        <SignOutButton>Sign Out</SignOutButton>
      </SignedIn>
      <SignedOut>
        <SignInButton>Sign In</SignInButton>
      </SignedOut>
    </header>

    <main>
      <SignedIn>
        <User>
          <template #default="{ user }">
            <div class="user-profile">
              <img
                v-if="user.picture"
                :src="user.picture"
                :alt="user.name || 'User avatar'"
                style="width: 80px; height: 80px; border-radius: 50%"
              />
              <h2>Welcome, {{ user.name || user.username }}!</h2>
              <div class="user-details">
                <p><strong>Email:</strong> {{ user.email }}</p>
                <p><strong>First Name:</strong> {{ user.given_name }}</p>
                <p><strong>Last Name:</strong> {{ user.family_name }}</p>
              </div>
            </div>
          </template>
        </User>
      </SignedIn>
    </main>
  </{{ProductName}}Provider>
</template>
```

## Run Your App

Start the development server:

<CodeGroup>
  <CodeBlock lang="bash" label="npm">
    npm run dev
  </CodeBlock>

  <CodeBlock lang="bash" label="Yarn">
    yarn dev
  </CodeBlock>

  <CodeBlock lang="bash" label="pnpm">
    pnpm run dev
  </CodeBlock>
</CodeGroup>

Visit your app at [http://localhost:5173](http://localhost:5173)

:::note Test credentials
You'll need a user to sign in with. If you haven't created one yet, open <ConsoleUrl />, navigate to **Users**, and add a test user with an email and password.
:::

:::tip Success
You should see the sign-in button. Click it, you'll be redirected to the <ProductName />-hosted sign-in page. Authenticate with your test user, and you will land back in your app with the user profile displayed.
:::
