## Run <ProductName />

Start a local ThunderID instance. Pick the method that works best for you:

<RunThunderID />

Once it's running, the console is available at [https://localhost:8090](https://localhost:8090).

## Create an Application

Open the Console at <ConsoleUrl />, navigate to **Applications**, and click **Add Application**:

1. Under **Technology**, select **Vue**.
2. Enter a name (e.g. `My Vue App`) and create an application. The rest of the settings can stay at their defaults.
3. Copy the **Client ID** from the **General** tab. You'll need it when configuring the SDK.
4. Under **General**, add `http://localhost:5173` to the list of **Authorized Redirect URIs**.

## Create a Vue App

Create your new Vue 3 app using Vite:

<CodeGroup>
  <CodeBlock lang="bash" label="npm">
    npm create vite@latest my-vue-app -- --template vue
    cd my-vue-app
    npm install
  </CodeBlock>

  <CodeBlock lang="bash" label="Yarn">
    yarn create vite my-vue-app --template vue
    cd my-vue-app
    yarn install
  </CodeBlock>

  <CodeBlock lang="bash" label="pnpm">
    pnpm create vite my-vue-app --template vue
    cd my-vue-app
    pnpm install
  </CodeBlock>
</CodeGroup>

## Install @thunderid/vue

Install the <ProductName /> Vue SDK in your project:

<CodeGroup>
  <CodeBlock lang="bash" label="npm">
    npm install @thunderid/vue
  </CodeBlock>

  <CodeBlock lang="bash" label="Yarn">
    yarn add @thunderid/vue
  </CodeBlock>

  <CodeBlock lang="bash" label="pnpm">
    pnpm add @thunderid/vue
  </CodeBlock>

  <CodeBlock lang="bash" label="Bun">
    bun add @thunderid/vue
  </CodeBlock>
</CodeGroup>

## Register the Plugin

Register `ThunderIDPlugin` in `src/main.js`:

```js title="src/main.js" {2,6-8} showLineNumbers
import { createApp } from 'vue'
import { ThunderIDPlugin } from '@thunderid/vue'
import './style.css'
import App from './App.vue'

const app = createApp(App)
app.use(ThunderIDPlugin)
app.mount('#app')
```

## Build with ThunderID Components

Update `src/App.vue` with ThunderID authentication components:

:::warning Configuration
Replace `<your-client-id>` with the Client ID you obtained when creating the application in <ProductName />.
:::

```vue title="src/App.vue" showLineNumbers
<script setup>
import {
  SignedIn,
  SignedOut,
  SignInButton,
  SignOutButton,
  UserDropdown,
} from '@thunderid/vue'
</script>

<template>
  <ThunderIDProvider
    client-id="<your-client-id>"
    base-url="https://localhost:8090"
  >
    <section id="center">
      <SignedIn>
        <UserDropdown />
      </SignedIn>
      <SignedOut>
        <SignInButton>Sign In</SignInButton>
      </SignedOut>
    </section>
  </ThunderIDProvider>
</template>
```

## Run Your App

Start the development server:

<CodeGroup>
  <CodeBlock lang="bash" label="npm">
    npm run dev
  </CodeBlock>

  <CodeBlock lang="bash" label="Yarn">
    yarn dev
  </CodeBlock>

  <CodeBlock lang="bash" label="pnpm">
    pnpm run dev
  </CodeBlock>
</CodeGroup>

:::note Test credentials
You'll need a user to sign in with. If you haven't created one yet, open <ConsoleUrl />, navigate to **Users**, and add a test user with an email and password.
:::

:::tip Success
Visit your app at 🌐 [http://localhost:5173](http://localhost:5173)

You should see the `<SignInButton />`. Click it to be redirected to the <ProductName />-hosted sign-in page. Login with your test user, then return to your app with the user dropdown displayed.
:::
