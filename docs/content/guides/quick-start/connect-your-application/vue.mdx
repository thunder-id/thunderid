## Create an Application in <ProductName />

Before integrating <ProductName /> with your Vue app, you need to create an application in <ProductName />.

### Using the <ProductName /> Console

1. Sign into the <ProductName /> Console at [https://localhost:8090/console](https://localhost:8090/console)
2. Navigate to **Applications** → **New Application**
3. Provide the following details:
   - **Name**: `my-vue-app`
   - **Type**: Single Page Application (SPA)
   - **Authorized Redirect URL**: `http://localhost:5173`

:::info
The authorized redirect URL determines where <ProductName /> sends users after they successfully sign in. For this guide, use `http://localhost:5173`, as the Vite development server runs at this URL by default.
:::

### Using the <ProductName /> API

You can also create an application using the <ProductName /> API. First, [obtain a system API token](/docs/next/guides/getting-started/get-thunderid#obtain-system-api-token), then:

```bash
curl -kL -X POST https://localhost:8090/applications \
  -H 'Authorization: Bearer <your-system-token>' \
  -H 'Content-Type: application/json' \
  -d '{
    "name": "my-vue-app",
    "inboundAuthConfig": [{
      "type": "oauth2",
      "config": {
        "grantTypes": ["authorization_code", "refresh_token"],
        "responseTypes": ["code"],
        "redirectUris": ["http://localhost:5173"],
        "tokenEndpointAuthMethod": "none",
        "publicClient": true,
        "pkceRequired": true
      }
    }]
  }'
```

:::warning Important
Note down the **Client ID** from the response. You'll need it to configure your Vue app.
:::

## Register the Plugin and Add <ProductName />Provider

The `@thunderid/vue` SDK provides a Vue plugin that registers the <ProductName />Provider component globally. Register the plugin in `main.js`, then wrap your root component with <ProductName />Provider in `App.vue`.

**Step 1:** Update `src/main.js` to register the plugin:

```js title="src/main.js" showLineNumbers
import { createApp } from 'vue'
import { {{ProductName}}Plugin } from '@thunderid/vue'
import App from './App.vue'
import './style.css'

const app = createApp(App)
app.use({{ProductName}}Plugin)
app.mount('#app')
```

**Step 2:** Wrap your application with <ProductName />Provider in `src/App.vue`:

```vue title="src/App.vue" showLineNumbers
<template>
  <{{ProductName}}Provider
    client-id="<your-client-id>"
    base-url="https://localhost:8090"
  >
    <!-- your app content goes here -->
  </{{ProductName}}Provider>
</template>

<script setup>
</script>
```

:::warning Configuration
Replace `<your-client-id>` with the Client ID you obtained when creating the application in <ProductName />.
:::

### Configuration Parameters

| Parameter | Description |
|-----------|-------------|
| `client-id` | The Client ID from your <ProductName /> application |
| `base-url` | Your <ProductName /> instance URL (e.g., `https://localhost:8090`) |

## Add Sign-In And Sign-Out

The <ProductName /> Vue SDK provides `SignInButton` and `SignOutButton` components along with `SignedIn`, `SignedOut`, and `Loading` components for conditional rendering.

Replace the content of `src/App.vue` with:

```vue title="src/App.vue" showLineNumbers
<script setup>
import {
  SignedIn,
  SignedOut,
  SignInButton,
  SignOutButton,
  Loading,
} from '@thunderid/vue'
</script>

<template>
  <{{ProductName}}Provider
    client-id="<your-client-id>"
    base-url="https://localhost:8090"
  >
    <Loading>
      <div>Loading authentication...</div>
    </Loading>

    <header>
      <h1>{{ProductName}} Auth Demo</h1>
      <SignedIn>
        <SignOutButton>Sign Out</SignOutButton>
      </SignedIn>
      <SignedOut>
        <SignInButton>Sign In</SignInButton>
      </SignedOut>
    </header>
  </{{ProductName}}Provider>
</template>
```

## Display User Profile Information

Use the `User` component with a scoped slot to access and display user profile information:

```vue title="src/App.vue" showLineNumbers
<script setup>
import {
  SignedIn,
  SignedOut,
  SignInButton,
  SignOutButton,
  Loading,
  User,
} from '@thunderid/vue'
</script>

<template>
  <{{ProductName}}Provider
    client-id="<your-client-id>"
    base-url="https://localhost:8090"
  >
    <Loading>
      <div>Loading authentication...</div>
    </Loading>

    <header>
      <h1>{{ProductName}} Auth Demo</h1>
      <SignedIn>
        <SignOutButton>Sign Out</SignOutButton>
      </SignedIn>
      <SignedOut>
        <SignInButton>Sign In</SignInButton>
      </SignedOut>
    </header>

    <main>
      <SignedIn>
        <User>
          <template #default="{ user }">
            <div class="user-profile">
              <img
                v-if="user.picture"
                :src="user.picture"
                :alt="user.name || 'User avatar'"
                style="width: 80px; height: 80px; border-radius: 50%"
              />
              <h2>Welcome, {{ user.name || user.username }}!</h2>
              <div class="user-details">
                <p><strong>Email:</strong> {{ user.email }}</p>
                <p><strong>First Name:</strong> {{ user.given_name }}</p>
                <p><strong>Last Name:</strong> {{ user.family_name }}</p>
              </div>
            </div>
          </template>
        </User>
      </SignedIn>
    </main>
  </{{ProductName}}Provider>
</template>
```

## Run Your App

Start the development server:

<CodeGroup>
  <CodeBlock lang="bash" label="npm">
    npm run dev
  </CodeBlock>

  <CodeBlock lang="bash" label="Yarn">
    yarn dev
  </CodeBlock>

  <CodeBlock lang="bash" label="pnpm">
    pnpm run dev
  </CodeBlock>
</CodeGroup>

Visit your app at [http://localhost:5173](http://localhost:5173)

:::tip Success
You should see the sign-in button. Click it, you'll be redirected to the <ProductName />-hosted sign-in page. Authenticate with the test user you created in step 2, and you will land back in your app with the user profile displayed.
:::
