# Prerequisites

Before you begin, ensure you have the following installed on your system:

## Development Tools

### Core Development

- **[Git](https://git-scm.com/downloads)** - Version control system
- **[Go](https://golang.org/doc/install)** - Version 1.26 or higher
- **[Node.js](https://nodejs.org/en/download/)** - `LTS` version is required
- **[pnpm](https://pnpm.io/installation)** - Latest version

### Frontend Development (If Applicable)

- **[ESLint VSCode Extension](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)** - For linting support in VSCode
- **[Prettier VSCode Extension](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)** - For code formatting in VSCode
- **[React Developer Tools](https://react.dev/learn/react-developer-tools)** - For inspecting React component trees and state in the browser

## Setup the Repository

### Fork the Repository

Start by forking the WSO2 <ProductName /> repository to your GitHub account.

### Clone the Repository

Clone your forked repository to your local machine.

```bash
git clone https://github.com/<your-username>/thunder.git
cd thunder
```

## 💻 macOS / Linux Prerequisites

`sqlite3` and `openssl` are pre-installed on most systems. If not, install via `brew` (macOS) or `apt`/`yum` (Linux).

## 💻 Windows Prerequisites

If you are contributing from a Windows environment, please ensure the following tools are installed before starting development.

### SQLite3

SQLite3 is required for local development and build processes.

**Install using:**

- `winget install SQLite.SQLite`
- or `choco install sqlite`

**Verify installation:**

```powershell
sqlite3 --version
```

### PowerShell 7

PowerShell 7 is required to run the PowerShell-based build and setup scripts.

**Install using:**

- `winget install Microsoft.PowerShell`
- or `choco install powershell-core`

**Verify installation:**

```powershell
pwsh --version
```
